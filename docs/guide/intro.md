# 入门

这里是学习文档的第一篇内容。

## 写作方式

- 在 `docs/` 目录下新增 Markdown 文件。
- 在 `docs/.vitepress/config.js` 中维护导航和侧边栏。
- 图片等静态资源可以放在 `docs/public/` 目录。

## 常用命令

```powershell
scripts\dev.cmd
scripts\build.cmd
scripts\preview.cmd
```

本地开发地址：

```text
http://localhost:5188
```

构建预览地址：

```text
http://localhost:4188
```

## 发布目录

构建完成后，发布下面这个目录即可：

```text
docs\.vitepress\dist
```
